# Introduction

Firefiles is an open-source alternative to Dropbox that allows you to bring your own cloud provider and lets you manage files across all providers seamlessly and without any hassle.

![firefiles-demo](/firefiles-demo3.png)

If you have any suggestions or feature requests, please let us know by creating an issue at our [Github](https://github.com/faisalsayed10/firefiles) repository / Reaching out to us on [Twitter](https://twitter.com/tryfirefiles) / [Email](mailto://firefiles@fayd.me).
